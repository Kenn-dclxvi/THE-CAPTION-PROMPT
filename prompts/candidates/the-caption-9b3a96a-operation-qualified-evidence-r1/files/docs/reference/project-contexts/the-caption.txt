<feature_decision_map>
  <rule topic="daily_mail">v4 daily mail changes target src/app/v4_engine.py, src/app/renderer/, and v4 view models.</rule>
  <rule topic="ledger">Asset valuation and canonical totals belong to UniversalIngester, ledger_schema, and ShadowLedger finalization. Renderer must not recompute ledger facts.</rule>
  <rule topic="collection">data/collection/market_units.csv is v4 SSOT A. collection_engine.py and collection_main.py have been retired to legacy/v3/ and are not the v4 canonical path.</rule>
  <rule topic="monthly">Monthly v4 chronicle changes target monthly_curator, daily_metrics inputs, and v4_chronicle template. Portfolio Audit belongs to monthly flow, not daily flow.</rule>
  <rule topic="ai_prompt">Curator/Chronicle prompt changes target src/config/prompts.py and docs/reference/prompts.md. Keep system instructions separate from user data tags.</rule>
  <rule topic="view">Display-only changes belong to src/app/renderer/, including templates and view-model modules. View changes must not alter ledger generation or SSOT semantics.</rule>
</feature_decision_map>
