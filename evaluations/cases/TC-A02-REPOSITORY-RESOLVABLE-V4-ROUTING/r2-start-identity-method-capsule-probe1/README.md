# TC-A02-REPOSITORY-RESOLVABLE-V4-ROUTING r2-start-identity-method-capsule-probe1

## 目的

A02 r2の成果条件、authority、permission、fixtureを変えず、開始identity operationだけへ局所method capsuleを提示する診断revisionである。

開始identityは4つの個別commandを同一model stepで取得する。terminal後はworkspace、branch、HEAD、statusだけを渡し、method、tool grouping、invocation状態、raw outputを後続のcanonical target解決へ渡さない。後続operationのmethodは未指定のままとする。

このrevisionの結果はA02 r2と互換KPI比較しない。Candidate55 / Candidate60の同一条件probeだけに使用する。
