# TC-F10-MONTHLY-FORMAT-TEST-REVIEW r3-method-capsule-probe1

## 目的

F10 r3の成果条件、authority、permission、fixtureを変えず、固定差分review operationだけへ局所method capsuleを提示する診断revisionである。

methodは、列挙済みの相互非依存なread-only入力を同一model stepで取得する。operation外へ渡す情報はreview findingとfinding countだけとし、tool grouping、失敗状態、raw outputは渡さない。

このrevisionの結果はF10 r3と互換KPI比較しない。Candidate55 / Candidate60の同一条件probeだけに使用する。
